# Outreach lists — named targets, contact paths, first-touch copy

Growth/Launch lane · 2026-07-20 · companion to
`spec/socials-operating-plan.md` (weeks reference these IDs). **Every draft
here is send-ready but sends only from the founder's hands, personalized.**

Verification rule (per the session brief): every named target below was
checked by web search on 2026-07-20; where a direct email/DM path could not
be verified, the entry says **"find via …"** instead of inventing one. Before
sending, glance at the target's site — boards turn over.

The hook, everywhere: **the Founding League offer** — PIGL is №1; the first
~5–10 leagues play free forever, named on the founding wall. It lives in DMs,
calls, and flyers — never on the app's front door, and never beside pricing
talk (pricing is parked; no numbers in public, ever).

Money language in every draft (D39): *"the app keeps the ledger — money
moves between friends, not through the app."* Never betting/gambling/wager.

## 4. Sequencing + hygiene

- Warm before the ask where the doc says so (N4/N5: subscribe + one genuine
  reply in W4; the ask lands W6 with the store link live).
- One personalized send per target — never a blast; log each send + reply
  in the Monday report (the outreach line feeds "leagues created" and
  nothing else; installs are not the point).
- Every reply that smells like an organizer → offer the white-glove call
  (the highest-leverage founder hour; GTM §12).
- If a target goes quiet: one follow-up after 10–14 days, then done — the
  register is "friend who runs a real league," and that guy doesn't nag.
