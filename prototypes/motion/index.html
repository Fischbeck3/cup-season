Seven clickable concepts — one entry-choreography evolution, six bold moments. Every one: tokens copied in, no libraries, transform/opacity only, reduced-motion designed per concept (toggle it on each page), gold spent only on what's earned. Full cards + build order in spec/motion-dossier.md.
